## 最大子序和 

先从最简单的说起，有一个整数数组 nums，找到里面和最大的那个连续子数组，并返回最大和。

这道题非常简单，对于一个连续子数组的最大和，假设为 dp[i],则
dp[i] = max(dp[i] + nums[i],nums[i])。

核心代码如下：
```C++
for(int i = 1;i < len;i++) {
    dp[i] = max(dp[i - 1] + nums[i],nums[i]);
    res = max(dp[i],res);
}
```
